Toast 轻提示

对操作结果的轻量级反馈，无需用户操作即可自行消失。

何时使用

适用于页面内容的变化不能直接反应操作结果时使用。

Toast

指令式 API

Toast 只支持指令式调用。

@inject ToastService ToastService;
ToastService.Show(new ToastOption(){});

ToastService.Show

Show 方法支持传入一个 ToastOption 对象，它包含了以下这些属性：

Property Type Default Description
AfterCloseFunc<ValueTask>Toast 完全关闭后的回调
ContentStringOrRenderFragmentToast 文本内容
DurationMsInt322000提示持续时间，若小于等于 0 则不会自动关闭
IconNullable<OneOf<ToastIconType, RenderFragment>>Toast 图标，支持枚举 ToastIconType 或 RenderFragment
MaskClassClassBuilderToast 遮罩类名
MaskClickableBooleanTrue是否允许背景点击
MaskStyleStyleBuilderToast 遮罩样式
PositionToastPositionCenter垂直方向显示位置

同一时间只允许弹出一个轻提示，新出现的 Toast 会将之前正在显示中的 Toast 挤掉。

Show 方法返回一个 ToastRef 对象，可以通过 ToastRef.Close 关闭显示中的 Toast 对象.

ToastService.Clear

关闭所有显示中的 Toast。

ToastService.Config

全局配置，支持配置 duration、position 和 maskClickable。

public void Config(
        int? durationMs = null,
        ToastPosition? position = null,
