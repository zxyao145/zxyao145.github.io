Space 间距

元素排列中保持相同的宽度。

何时使用

适用于多个元素按照水平或垂直方向保持相同的间距。

API

属性

Property Type Default Description
AlignStringstart | end |center |baseline
JustifyStringCross axis alignment
BlockBooleanFalseWhether to render as block level elements
WrapBooleanFalseWhether to automatically wrap, only valid when horizontal.
DirectionBmDirectionhorizontalSpacing direction
OnClickEventCallback<MouseEventArgs>Click event of Space.

属性 说明 默认值

--space-gap-vertical 水平方向的间距大小 var(--space-gap)
--space-gap-horizontal 垂直方向的间距大小 var(--space-gap)
